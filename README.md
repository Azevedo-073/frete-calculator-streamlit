# 🚛 Calculadora de Frete

Aplicação desenvolvida para automatizar o cálculo de fretes com base em regras logísticas reais, considerando tabelas por cliente, tipo de veículo, operação (CIF/FOB) e rotas.

## 📌 Objetivo

No dia a dia logístico, o cálculo de frete pode ser complexo devido à quantidade de variáveis envolvidas, como:
- Diferentes tabelas por cliente
- Tipos de veículos
- Regras específicas por operação
- Distância e rota

Essa aplicação foi criada para reduzir o trabalho manual e aumentar a precisão nos cálculos.

## ⚙️ Funcionalidades

- Seleção de origem e destino
- Aplicação automática de regras de cálculo
- Cálculo baseado em tabelas logísticas
- Simulação de cenários de frete

## 🛠️ Tecnologias utilizadas

- Python
- Streamlit
